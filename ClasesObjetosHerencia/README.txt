CLASESOBJETOSHERENCIA
Aplicacion que solicita un lado, el usuario podra escoger que opcion del siguiente menu:

-Area de un cuadrado(realiza operacion)

-Perimetro de un cuadrado

-Volumen de un cubo

-Perimetro de un cubo

-Cancelar(reinicia aplicacion solicitando lado)

-Salir(termina aplicacion)

Una vez hecha la operacion se podra regresar al menu y hacer una nueva operacion o podra reiniciar la aplicacion
para ingresar un nuevo lado
